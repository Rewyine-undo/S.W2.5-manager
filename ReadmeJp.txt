# S.W 2.5 manager
 
 
 ソード・ワールド 2.5 のキャラクターデータを管理できるアプリケーションです。
 現在ルールブックⅠ～Ⅲのデータに対応しています。
 その他サプリメントのデータには対応していません。
 
# DEMO
 
 
 デモ紹介なんてねぇよ
 
 
# Features
 
 
 煩雑な能力値、経験値周りの処理を自動で行ってくれます。
 また、装飾品の記録や上級戦闘における位置関係の描画も行えます。
 
# Requirement
 
 
 wxWidgets-3.2.6
 nlohmann/json
 
 
# Installation
 <for max/Linux>
 https://github.com/Rewyine-undo/S.W2.5-manager
 このgithubのリポジトリをクローンし、build process.txtを元にビルドを行って下さい。
 
 <for windows>
 現状、ライブラリ及び実行ファイルをコンパイルしてから配布しているため、Windows os上での使用であれば
 ライブラリのインストール、コンパイル等は不要です。
 「S.W 2.5 manager_(更新日).exe」(実行ファイル)を起動するだけで使用できます。
 
 
# Usage
 
 
 <<キャラクター新規作成の場合>>
     1. 「能力作成」タブを選択する。
     2. 名前(後に変更可)、種族、生まれの能力値を設定する。
     3. 「作成」ボタンを押す。
     4. 設定したキャラクターの能力値作成が3回だけ試行されるので、表示される情報を見て
        気に入った試行があれば、その試行の右にある「この能力値で新規作成」ボタンを押す。
        気に入った試行がなければ、「閉じる」ボタンを押す。
     5. 自動で「データ表示」タブに移動するため、キャラクターデータの設定を続行する。
 
 
 <<作成したキャラクターデータ(.jsonファイル)を保存する場合>>
     1. 左上の「ファイル」メニューを選択する。
     2. 「名前を付けてファイルを保存」を選択する。
     3.任意のディレクトリにファイルを保存する。
 
     またはCtrl+shift+Sでもファイル保存画面に移動できます。


 <<作成したキャラクターデータ(.jsonファイル)を読み込む場合>>
     1. 左上の「ファイル」メニューを選択する。
     2. 「ファイルを読み込む」を選択
     3. 任意のディレクトリからファイルを選択する。
 
     またはCtrl+Fでもファイル保存画面に移動できます。


 <<キャラクターデータ(.jsonファイル)を上書き保存する場合>>
     1. 左上の「ファイル」メニューを選択する。
     2. 「ファイルを上書き保存」を選択する。
     
 
     またはCtrl++Sでも上書き保存できます。
     ただし、この操作以前に「ファイルの保存」もしくは「ファイルの読み込み」を行っていなかった場合、
     自動で「名前を付けてファイルを保存」の画面に移動します。


 <<上級戦闘を管理する場合>>
     1. 「上級戦闘(GM向け)」タブを選択する。
     2. 左のテキストボックスに味方キャラクターの名前や識別子を入力する。
     3. 画面上部に駒が表示され、駒の位置がテキストボックス右に赤字で表示される。
     4. スライダーを操作して駒の位置を管理する。
     5. 味方キャラクターが複数いる場合は、画面中央の「味方を追加」ボタンを押す。
     6. 新たなキャラクター分のテキストボックスが生成されるため、同様に入力する。
 
     同様の操作を敵側のキャラクターに対しても行い、戦闘場面を再現します。
     キャラクターが倒された、などで戦闘から離脱する場合、スライダー右の「削除」ボタンを押すことで
     駒を画面上部の数直線から排除されます。



# Note
 
 
 * Linux os上での動作は検証していないため、いかなる環境でもその動作は保証しません。

 * 基本的に数値のみが入るボックスに数値以外の要素を入力した状態でファイルを保存しようとすると、
   エラーが発生する可能性があります。

 * 判定パッケージ等の武具のデータを扱う能力値は、記入項目欄の中で最大の数値を選択して表示しているため、
   マイナス補正を持つ武器の特性は正常に反映されません。
   マイナス補正を持つ武器をメインに扱う場合はこの仕様を念頭に置いてください。

 * 常時特技による能力値の補正機能に関しては、回避行動のランクはローマ数字(Ⅰ,Ⅱ)で入力し、
   武器習熟のランクは半角大文字(A,S)で入力してください。
   その他の記法の場合は補正が正常に反映されません。

 * 今後は、不定期に以下の要素を実装していきます。
     ・ 選択習得した練技、呪歌&終律、騎芸、賦術を記述する欄の追加
     ・ 種族特徴の強化の効果の自動適用機能の実装
     ・ 複数のキャラクターデータ表示の同時展開
     ・ 立ち絵表示欄の追加
 
 * 確認された不具合一覧
     ・ 前バージョンで作成されたファイルを現行バージョンで読み込むと、「所持アイテムなど」欄のデータが消失する。
     ・ 「上級戦闘(GM向け)」タブにおいて、味方の入力欄数が0の状態で敵の入力欄を追加すると、敵の入力欄が画面の
         左側に移動する。
 
 以後、不定期に不具合を修正していきます。
